# Omido Positioning and Information Architecture

## Purpose
Define a stronger premium market position and translate it into a sharper website structure.

## Use this skill when
- redefining homepage structure
- improving clarity
- building a page narrative
- restructuring services or proof
- improving CTA flow

## Focus
- target audience
- positioning statement
- service prioritization
- homepage narrative sequence
- page roles
- proof placement
- objection handling
- CTA logic
- contact path clarity

## Required output
Provide:
1. positioning direction
2. ideal audience assumptions
3. revised sitemap or page-purpose map
4. homepage structure
5. section-by-section rationale
6. CTA strategy
7. trust architecture

## Standard
A premium website must guide a buying decision, not just display information.
