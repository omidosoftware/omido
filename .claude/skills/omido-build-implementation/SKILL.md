# Omido Build Implementation

## Purpose
Translate redesign strategy into code architecture and implementation steps.

## Use this skill when
- planning component structure
- implementing redesigned pages
- standardizing frontend architecture
- aligning UI and engineering

## Focus
- page composition
- component structure
- section modularity
- responsive behavior
- animation architecture
- semantic HTML
- accessibility
- maintainability
- performance budgets

## Required output
Provide:
1. component map
2. page implementation plan
3. responsive rules
4. motion implementation plan
5. accessibility requirements
6. performance constraints

## Standard
Build for long-term maintainability, not demo-only impressiveness.
