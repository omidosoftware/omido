# Omido Site Audit

## Purpose
Run a ruthless audit of the current Omido website and identify what prevents it from feeling premium, highly credible, and high-converting.

## Use this skill when
- starting the redesign
- evaluating the current website
- reviewing weak trust signals
- identifying gaps in hierarchy, content, and UX

## Audit checklist
Review:
- homepage
- services page
- portfolio page
- about page
- contact page
- navigation
- CTA flow
- mobile clarity
- trust density
- proof quality
- visual hierarchy
- spacing rhythm
- typography authority
- premium perception
- differentiation
- lead quality fit

## Required output
Provide:
1. strengths
2. weaknesses
3. strategic gaps
4. visual/UX gaps
5. conversion gaps
6. mobile-specific gaps
7. top 10 redesign priorities

## Standard
Do not be polite.
Do not be vague.
Be commercially useful.
