# Omido Premium UI System

## Purpose
Create a premium UI system that feels intentional, refined, modern, and expensive.

## Use this skill when
- defining typography
- defining spacing
- building layouts
- standardizing components
- improving premium feel

## Focus
- typography scale
- line length
- spacing scale
- grid system
- section rhythm
- card hierarchy
- buttons
- inputs
- labels
- badges
- icon use
- visual consistency
- contrast and dark-mode sensibility

## Required output
Provide:
1. typography rules
2. spacing rules
3. grid rules
4. component rules
5. section rhythm guidance
6. premium visual anti-patterns to avoid

## Standard
Premium does not mean decorative excess.
Premium means control, clarity, and refinement.
