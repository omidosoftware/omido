# Omido Performance and Accessibility

## Purpose
Ensure the redesigned site remains fast, stable, inclusive, and production-ready.

## Use this skill when
- reviewing the redesign
- testing motion impact
- checking responsiveness
- improving accessibility
- protecting Core Web Vitals

## Focus
- semantic structure
- keyboard access
- focus states
- contrast
- reduced motion
- image optimization
- font loading
- JS weight
- animation cost
- LCP, CLS, INP risk

## Required output
Provide:
1. accessibility audit
2. performance risk audit
3. Core Web Vitals risks
4. specific fixes
5. launch-readiness verdict

## Standard
Premium websites must feel good and perform well.
