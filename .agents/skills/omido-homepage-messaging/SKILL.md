# Omido Homepage Messaging

## Purpose
Redesign the homepage messaging so it feels premium, commercially credible, clear, and differentiated from generic AI/software agencies.

## Use this skill when
- rewriting the homepage
- replacing weak hero messaging
- removing stack-led messaging
- improving narrative flow
- improving clarity and trust in the first 10 seconds

## Core Rules
- Do not lead with tool names
- Do not create a homepage section that mainly lists technologies
- Do not sound like a generic AI agency
- Do not use buzzword-heavy filler copy

## Focus
- hero headline
- hero subheadline
- homepage section order
- primary CTA framing
- trust signals
- differentiators
- “how we build” or “our priorities” messaging
- mobile-first clarity
- premium tone
- immediate credibility

## Required output
Provide:
1. homepage messaging diagnosis
2. 3 stronger homepage directions
3. hero headline options
4. hero subheadline options
5. recommended homepage section order
6. replacement for “our technology” messaging
7. CTA recommendations
8. strongest final direction

## Standard
The homepage must make Omido feel more valuable, more credible, and easier to trust fast.
